Background-color

Определяет цвет фона элемента

фон по умолчанию - прозрачный

.block{ background: transparent; } или .block{ background-color: transparent; }

цветной фон

.block{ background: #18b5a4; } или .block{ background-color: #18b5a4; }

Цветной (RGB)

.

.block{ background: rgb(24,181,164); } или .block{ background-color: rgb(24,181,164); }

Цветной (RGBA) прозрачный

.block{ background: rgba(24,181,164,0.5); } или .block{ background-color: rgba(24,181,164,0.5); }

Background-image

Устанавливает фоновое изображение для элемента либо задает градиентную заливку

Картинка как фон, по умолчанию стоит функция репит

.block{ background: url('img/background/avatar.jpg'); } или .block{ background-image: url('img/background/avatar.jpg'); }

Linear - последовательный переход между двумя и более цветами линейно

background-image: linear-gradient(55.95deg, #FF3F3A 0%, #F75E05 100%);

Генератор градиентов https://cssgradient.io/

Linear - последовательный переход между двумя и более цветами круговым вариантом

background-image: radial-gradient(circle at center, red 0, blue, green 100%);

Генератор градиентов https://cssgradient.io/

Background-repeat

Определяет, как будет повторяться фоновое изображение

Фоновое изображение повторяется по горизонтали и вертикали.

.block{ background: url('img/background/avatar.jpg') repeat; } или .block{ background-image: url('img/background/avatar.jpg'); background-repeat: repeat; }

Устанавливает одно фоновое изображение в элементе без его повторений

.block{ background: url('img/background/avatar.jpg') no-repeat; } или .block{ background-image: url('img/background/avatar.jpg'); background-repeat: no-repeat; }

Фоновый рисунок повторяется только по горизонтали.

.block{ background: url('img/background/avatar.jpg') repeat-x; }

.block{ background-image: url('img/background/avatar.jpg'); background-repeat: repeat-x; }

Фоновый рисунок повторяется только по вертикали.

.block{ background: url('img/background/avatar.jpg') repeat-y; }

.block{ background-image: url('img/background/avatar.jpg'); background-repeat: repeat-y; }

Изображение повторяется столько раз, чтобы полностью заполнить область; если это не удаётся, между картинками добавляется пустое пространство.

.block{ background: url('img/background/avatar.jpg') space; }

.block{ background-image: url('img/background/avatar.jpg'); background-repeat: space; }

Изображение повторяется так, чтобы в области поместилось целое число рисунков; если это не удаётся сделать, то фоновые рисунки масштабируются.

.block{ background: url('img/background/avatar.jpg') round; }

.block{ background-image: url('img/background/avatar.jpg'); background-repeat: round; }

Background-position

Задает начальное положение фонового изображения, где первое значение (проценты, px, слово) определяет расположение по горизонтали, второе - вертикали.

По умолчанию.

.block{ background: url('img/background/avatar.jpg') 0 0 no-repeat; }

.block{ background: url('img/background/avatar.jpg') top left no-repeat; }

По центру.

.block{ background: url('img/background/avatar.jpg') 50% no-repeat; }

.block{ background: url('img/background/avatar.jpg') center no-repeat; }

Слева.

.block{ background: url('img/background/avatar.jpg') 0 50% no-repeat; }

.block{ background: url('img/background/avatar.jpg') left no-repeat; }

В углу.

.block{ background: url('img/background/avatar.jpg') 100% 100% no-repeat; }

.block{ background: url('img/background/avatar.jpg') bottom right no-repeat; }

Позиция.

.block{ background: url('img/background/avatar.jpg') 25px 30px no-repeat; }

.block{ background-image: url('img/background/avatar.jpg'); background-repeat: no-repeat; background-position: 25px 30px; }

Background-attachment

Устанавливает, будет ли прокручиваться фоновое изображение вместе с содержимым элемента.

По умолчанию - Позволяет перемещаться фону вместе с окном браузера

.block{ background: url('img/background/bg.jpg') 0 0 no-repeat scroll; }

.block{ background-image: url('img/background/bg.jpg'); background-repeat: no-repeat; background-position: 0 0; background-attachment: scroll; }

fixed - Делает фоновое изображение элемента неподвижным, а так же устанавливает фоновое изображение относительно окна браузера

.block{ background: url('img/background/bg.jpg') 0 0 no-repeat fixed; }

.block{ background-image: url('img/background/bg.jpg'); background-repeat: no-repeat; background-position: 0 0; background-attachment: fixed; }

Фон фиксируется с учётом поведения элемента. Если элемент имеет прокрутку, то фон будет прокручиваться вместе с содержимым, но фон выходящий за рамки элемента остаётся на месте.

.block{ background: url('img/background/bg.jpg') 0 0 no-repeat local; }

.block{ background-image: url('img/background/bg.jpg'); background-repeat: no-repeat; background-position: 0 0; background-attachment: local; }

Background-size

Масштабирует фоновое изображение согласно заданным размерам. В цифрах, где первое значение (проценты, px) определяет ширину, второе - высоту. Либо специальным значением.

Размеры картинки по умолчанию

.block{ background: url('img/background/bg.jpg') 0 0/auto auto no-repeat; }

.block{ background-image: url('img/background/bg.jpg'); background-position: 0 0; background-repeat: no-repeat; background-size: auto auto; }

Размеры картинки по ширине блока

.block{ background: url('img/background/bg.jpg') 0 0/100% auto no-repeat; }

.block{ background-image: url('img/background/bg.jpg'); background-position: 0 0; background-repeat: no-repeat; background-size: 100% auto; }

Размеры картинки по высоте блока

.block{ background: url('img/background/bg.jpg') 0 0/auto 100% no-repeat; }

.block{ background-image: url('img/background/bg.jpg'); background-position: 0 0; background-repeat: no-repeat; background-size: auto 100%; }

Расстягиваем картинку на всю ширину и высоту блока

.block{ background: url('img/background/bg.jpg') 0 0/100% 100% no-repeat; }

.block{ background-image: url('img/background/bg.jpg'); background-position: 0 0; background-repeat: no-repeat; background-size: 100% 100%; }

Масштабирует изображение с сохранением пропорций таким образом, чтобы картинка целиком поместилась внутрь блока. То есть по меньшей стороне картинки

.block{ background: url('img/background/bg.jpg') 0 0/contain no-repeat; }

.block{ background-image: url('img/background/bg.jpg'); background-position: 0 0; background-repeat: no-repeat; background-size: contain; }

Масштабирует изображение с сохранением пропорций так, чтобы его ширина или высота равнялась ширине или высоте блока. То есть по большей стороне картинки

.block{ background: url('img/background/bg.jpg') 0 0/cover no-repeat; }

.block{ background-image: url('img/background/bg.jpg'); background-position: 0 0; background-repeat: no-repeat; background-size: cover; }

Устанавливаем конкретные размеры. Если установлено одно значение, оно задает ширину фона, второе значение принимается за auto. Пропорции картинки при этом сохраняются. Использование двух значений через пробел задает ширину и высоту фоновой картинки

.block{ background: url('img/background/bg.jpg') 0 0/200px 150px no-repeat; }

.block{ background-image: url('img/background/bg.jpg'); background-position: 0 0; background-repeat: no-repeat; background-size: 200px 150px; }

Установка нескольких фоновых изображений для блока

Свойство background позволяет задать несколько фоновых изображений одному блоку, причем со своими настройками позиционирования масштабирования и прокрутки. Указывать значения следует через запятую.

Цвет фона + одна картинка + не повторять

.block{ background: #18b5a4 url('img/background/avatar.jpg') no-repeat; }

.block{ background-color: #18b5a4; background-image: url('img/background/bg.jpg'); background-repeat: no-repeat; }

Две картинки, у каждой свои настройки. Каждая следующая ложится под предыдущую

.block{ background: url('img/background/avatar_3.jpg') left no-repeat, url('img/background/bg.jpg') 50%/cover no-repeat; }

.block{ background-image: url('img/background/avatar_3.jpg'),url('img/background/bg.jpg'); background-repeat: no-repeat; background-position: left,50%; background-size: auto auto, cover; }

Три картинки, у каждой свои настройки. Каждая следующая ложится под предыдущую

.block{ background: url('img/background/avatar_2.jpg') right/50px no-repeat, url('img/background/avatar_3.jpg') left no-repeat, url('img/background/bg.jpg') 50%/cover no-repeat; }

.block{ background-image: url('img/background/avatar_2.jpg'), url('img/background/avatar_3.jpg'),url('img/background/bg.jpg'); background-repeat: no-repeat; background-position: right, left, 50%; background-size: 50px, auto auto, cover; }

Четыре картинки, у каждой свои настройки. Каждая следующая ложится под предыдущую

.block{ background: url('img/background/avatar.jpg') 0 0/50px no-repeat, url('img/background/avatar_2.jpg') 100% 100%/50px no-repeat, url('img/background/avatar_3.jpg') 50% no-repeat, url('img/background/bg.jpg') 50%/cover no-repeat; }

.block{ background-image: url('img/background/avatar.jpg'), url('img/background/avatar_2.jpg'), url('img/background/avatar_3.jpg'),url('img/background/bg.jpg'); background-repeat: no-repeat; background-position:0 0, 100% 100%, 50%, 50%; background-size: 50px, 50px, auto auto, cover; }
